# 📝 Project 00 – To-Do List App

This is the first project in my **30 Days of JavaScript Projects** challenge.

## 🚀 Features

- ✅ Add tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 💾 Tasks are stored in **Local Storage** (persist after reload)
- 🎨 Clean and modern UI
